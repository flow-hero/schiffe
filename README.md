# Schiffe Versenken
***
Beschreibung:
Ein Programm mit welchem es möglich ist Schiffe Versenken zu spielen.
Der Nutzer (Spieler) wählt die Position seiner Schiffe mit verschiendener Länge.
Der Computer versucht nun diese Schiffe abzuschießen.
